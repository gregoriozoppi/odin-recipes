# Odin Recipes

A basic recipe website built as part of The Odin Project curriculum.

## Description

This project demonstrates fundamental HTML skills by creating a simple recipe website with multiple recipe pages.

## Features

- Homepage with links to individual recipes
- Recipe pages with ingredients and instructions
- Basic HTML structure and navigation

## Skills Demonstrated

- HTML structure and syntax
- Links and navigation between pages
- Lists (ordered and unordered)
- Images
- Text formatting

## How to Use

1. Clone this repository
2. Open `index.html` in your web browser
3. Click on recipe links to view different recipes

## Project Status

This project is part of The Odin Project Foundations course.
